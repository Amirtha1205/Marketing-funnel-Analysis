# 📊 Marketing Funnel Analysis

## 📌 Project Overview

This project analyzes e-commerce customer behavior using Python and Tableau.
The main objective of this project is to understand the customer journey in the marketing funnel and identify how users move from viewing products to adding them to the cart and making purchases.

The project helps businesses improve customer conversion rates and make data-driven decisions.

---

## 🚀 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Tableau
* Jupyter Notebook

---

## 📂 Dataset

Dataset used:
E-Commerce Behavior Dataset from Kaggle

Dataset link:https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store?

The dataset contains:

* Event Type
* Product ID
* Category ID
* Category Code
* Brand
* Price
* User ID
* User Session

Event types include:

* View
* Cart
* Purchase

## 📈 Key Insights

* Most users only viewed products without purchasing.
* Purchase events were much lower compared to views.
* Cart products showed higher average prices.
* Customer drop-off was high between the view and purchase stages.
* Average product price was around 291.

---

## 🏢 Business Recommendations

* Improve marketing campaigns to increase conversions.
* Provide discounts and personalized offers to customers.
* Simplify the checkout process to reduce drop-offs.
* Retarget users who added products to the cart but did not purchase.
* Monitor customer behavior regularly using dashboards.
